# AuditLens

AuditLens is a static Arabic financial dashboard for small businesses and accountants.

## Current MVP

- Manual finance data entry.
- Financial health score.
- KPI dashboard.
- Revenue and liquidity charts.
- Notes and 30-day action plan.
- Full printable report.
- Privacy and terms pages.
- Contact links for email, WhatsApp, and LinkedIn.
- Netlify-ready feedback and lead capture form.

## Free Deployment

The project is static. It can be deployed for free on Netlify, Cloudflare Pages, or GitHub Pages.

Recommended first launch: Netlify Free.

No build command is required. Publish directory: `.`
